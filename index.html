Hospital Management System

2025 Live Demo Git Repo

A full-stack Hospital Management System built with Next.js, Express, Node.js & MongoDB. Admins manage doctors, users, and appointment status, while users can book appointments, track their status, and message the admin for support. It also supports blood and organ donation registration to help patients in need. Built on the Next.js App Router with route groups for role-based layouts, secure JWT auth via server-side Route Handlers and HTTP-only cookies, plus Tailwind CSS, TanStack Toolkit, and Stripe payments for a fast, scalable experience.

This Project is made using Next.js, Express, Node.js & MongoDB.
